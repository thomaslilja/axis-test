## :construction: Getting started

Clone the repository:

```bash
git clone git@github.com:thomaslilja/axis-test.git
```

Install dependencies:

```bash
npm i
```

Configure AWS:

```bash
aws sso login --profile <your-profile>
```

Run it locally:

```bash
npm run dev
```
